# CipherVault Pro v3

A desktop cryptography application built with Python and Tkinter.

## Features
- 12 encryption algorithms
- Symmetric & Asymmetric cryptography
- File encryption/decryption
- QR Code generation
- PDF export
- Modern GUI

## Implemented Algorithms

### Classical Ciphers
- Caesar Cipher
- ROT13
- Reverse Cipher
- Vigenere Cipher
- Rail Fence Cipher
- Transposition Cipher

### Public Key Cryptography
- ECC
- ElGamal
- Rabin
- Paillier
- Schmidt-Samoa
- Knapsack

```bash
git clone YOUR_REPO_LINK
cd CipherVault-Pro
pip install -r requirements.txt
